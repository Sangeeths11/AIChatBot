/api/register - POST (registration)
/api/login - POST (login)
/api/users - GET (get all users)
/api/users - POST (register a user)
/api/users/{userId} - GET (get a specific user)
/api/users/{userId} - PATCH (update a user)
/api/users/{userId}/subjects - POST (add a subject for a user)
/api/users/{userId}/subjects - GET (get all subjects for a user)
/api/users/{userId}/subjects/{subjectId} - GET (get a specific subject with resources)
/api/users/{userId}/subjects/{subjectId}/documents - GET (get all document IDs for a subject)
/api/users/{userId}/subjects/{subjectId}/documents - POST (add a document to a subject)
/api/users/{userId}/subjects/{subjectId}/documents/{documentId} - GET (get a specific document)
/api/users/{userId}/subjects/{subjectId}/videos - GET (get all video IDs for a subject)
/api/users/{userId}/subjects/{subjectId}/videos/{videoId} - GET (get a specific video)
/api/users/{userId}/subjects/{subjectId}/videos/{videoId}/summary - GET (get the summary for a specific video)
